# 🤖 AI Paper Assistant — versione Hugging Face

App per esplorare, riassumere e interrogare paper scientifici da arXiv.

## 🧠 Stack
| Componente | Tecnologia |
|-------------|-------------|
| Frontend | Streamlit |
| Backend | FastAPI |
| LLM | Hugging Face Inference API |
| Hosting | Hugging Face Spaces |
| Database | SQLite |

## 🚀 Avvio locale
```bash
pip install -r requirements.txt
streamlit run app/streamlit_app.py
